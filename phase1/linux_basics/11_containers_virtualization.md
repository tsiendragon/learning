# 容器和虚拟化

## 概念的解释
容器和虚拟化技术用于在单一物理主机上运行多个独立的操作系统环境。

## 参数的解释
- **Docker**：容器化平台。
  - `docker run`：运行容器。
  - `docker pull`：拉取镜像。
- **VirtualBox**：虚拟机管理器。
  - `VBoxManage`：命令行管理工具。

## 实践的例子
1. 使用 `docker run hello-world` 运行一个简单的容器。
2. 使用 `docker pull ubuntu` 拉取 Ubuntu 镜像。
3. 使用 VirtualBox 创建和管理虚拟机。
