# 函数与模块

## 概念的解释
函数是可重用的代码块，模块是包含函数和变量的文件。

## 参数的解释
- **函数定义**：使用 `def` 关键字。
  - `def func_name():`：定义函数。
- **模块导入**：使用 `import` 关键字。
  - `import math`：导入数学模块。

## 实践的例子
1. 定义一个简单的函数：
   ```python
   def greet():
       print("Hello!")
   ```
2. 调用函数：
   ```python
   greet()
   ```
3. 导入模块并使用函数：
   ```python
   import math
   print(math.sqrt(16))
   ```
