# 基本语法

## 概念的解释
Python 的基本语法包括变量定义、数据类型、控制结构等。

## 参数的解释
- **变量**：用于存储数据。
  - `x = 5`：定义整数变量。
- **数据类型**：如整数、字符串、列表。
  - `type(x)`：获取变量类型。
- **控制结构**：如条件语句和循环。
  - `if`：条件判断。
  - `for`：循环。

## 实践的例子
1. 定义一个变量并输出其值：
   ```python
   x = 10
   print(x)
   ```
2. 使用 `if` 语句判断条件：
   ```python
   if x > 5:
       print("x is greater than 5")
   ```
3. 使用 `for` 循环遍历列表：
   ```python
   for i in [1, 2, 3]:
       print(i)
   ```
